Regresión lineal

Método estadístico que se utiliza para poder modelar la relación entre una variable dependiente y una o más variables independientes. En pocas palabras, comprende cómo cambia una variable en función de otra variable. El coeficiente de correlación (r) varía entre -1 y 1

5.1 Regresión y correlación

correlación: Es la que mide la fuerza y dirección de la asociación entre dos variables

r = 1: correlación positiva perfecta

r = 0: Sin correlación

r = -1: correlación negativa perfecta

Regresión: Modela la relación funcional entre una variable dependiente (y) y una o más variables independientes (x)

Estima valores de y para un valor dado de x

Se representa mediante una ecuación lineal (y = mx + b) o una ecuación más compleja

5.1.1 Diagrama de dispersión.

También se le conoce como gráfica de dispersión, gráfico de correlación o nube de puntos. Es una representación de la relación entre dos variables. Se utiliza para visualizar la posible existencia de una correlación entre dos variables cuantitativas

5.1.2 Regresión lineal simple.

Método estadístico que se utiliza para modelar la relación linela entre una variable dependiente (y) y una única variable independiente (x)

correlación

Medida estadística que indica la fuerza y dirección de la asociación entre dos variables cuantitativas. Nos ayuda a comprender si existe una relación entre dos variables y qué tan fuerte es esa relación

5.1.4 Determinación y análisis de los coeficientes de correlación y de determinación.

Los coeficientes de correlación y determinación son herramientas útiles para explorar la relación entre dos variables, pero no deben usarse como únicas medidas para determinar la fuerza o la importancia de la relación.

5.1.5 Distribución normal bidimensional

También conocida como distribución gaussiana bidimensional, distribución que describe la probabilidad conjunta de que dos variables aleatorias continuas, X e Y, tomen valores específicos.

5.1.6 Intervalos de confianza y pruebas para el coeficiente de correlación.

Un intervalo de confianza para el coeficiente de correlación (ρ) proporciona un rango de valores dentro del cual se encuentra el valor verdadero de ρ con un nivel de confianza específico (por ejemplo, 95%).

Interpretación: Si el valor verdadero de ρ se encuentra dentro del intervalo de confianza, podemos estar seguros (con un nivel de confianza del 95%) de que el valor muestral (r) es una buena estimación de ρ. Un intervalo de confianza más estrecho indica una mayor precisión en la estimación de ρ.

5.1.7 Errores de medición

los errores de medición son desviaciones entre el valor verdadero de una magnitud y el valor que se obtiene al medirla. Estos errores son inevitables en cualquier proceso de medición y pueden deberse a diversos factores.

Errores sistemáticos: Son errores constantes que afectan a todas las mediciones por igual.

Errores aleatorios: Son errores aleatorios que varían de una medición a otra. Se producen por causas como la precisión del instrumento de medición.
